ING.SOFTWARE

Tema 2

La ingeniería de software es la disciplina que se encarga de diseñar, desarrollar y mantener programas de software, así como la documentación asociada. Se basa en la aplicación de principios, metodologías, arquitecturas y patrones para crear programas que resuelvan problemas, mejoren la eficiencia y faciliten tareas. La ingeniería de software se aplica en una amplia variedad de ámbitos, como la investigación científica, la medicina, la banca, la logística, la fabricación, la empresa, el control del tráfico y la meteorología. Algunas de las funciones de un ingeniero de software son: Diseñar y crear sistemas informáticos, Mantener el software existente para optimizar su desempeño, Coordinar equipos de trabajo, Conocer nuevas técnicas para el desarrollo de programas informáticos, Administrar bases de datos. El proceso de elaboración del software se denomina ciclo de vida y está formado por cuatro etapas: Concepción, Elaboración, Construcción, Transición.
