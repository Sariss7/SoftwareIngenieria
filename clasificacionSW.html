INGENIERÍA DE SOFTWARE

Clasificación del software

El software es un código o un conjunto de instrucciones que le dice a una computadora o hardware cómo operar. El software suele ser genérico, pero también puede ser personalizado. El software genérico está abierto al mercado y el programador diseña sus especificaciones. Están diseñados principalmente para un amplio mercado de clientes. El software personalizado es un software cuyas especificaciones están diseñadas de acuerdo con una empresa u organización en particular. No es un operador en absoluto. Están diseñados principalmente para fines comerciales específicos.

1. Software del sistema: Interactúa directamente con el hardware de la computadora. Se ocupa principalmente de la gestión eficiente del sistema informático. Los desarrolladores lo utilizan para desarrollar nuevos programas de sistema y pueden hacerlos portables mediante el arranque. Depende de la máquina. El software del sistema se clasifica además en tres categorías: el sistema operativo, que actúa como una interfaz entre el usuario y el hardware y proporciona diferentes servicios a los usuarios. El segundo es el software de soporte del sistema que administra el hardware de manera más eficiente. El otro es el software de desarrollo del sistema que admite un entorno de desarrollo de programación para el usuario.

2. Software de aplicación: El software está diseñado para resolver problemas de los usuarios según sus requisitos. El software de aplicación puede ser genérico o personalizado. Además, clasificamos el software de aplicación en dos categorías: una es el software de propósito general que los usuarios utilizan para una amplia gama de tareas y proporciona numerosas funciones. Otro tipo de software es el software de propósito especial, diseñado específicamente para un propósito particular. Por ejemplo, los programas de usuario. El enfoque se centra en la aplicación, no en el sistema informático. Se ocupa principalmente de la solución de algunos problemas utilizando la computadora como herramienta.

3. Software de desarrollo: Dispositivo específico de cualquier equipo informático. Podría clasificarse como aplicado, pero debido a las particularidades de su aplicación se ha destacado como un tipo aparte. Su función principal es la de depuración, el ajuste y la reescritura del código del programa. Esto incluye compiladores, depuradores, traductores de alto nivel, editores, intérpretes y otras herramientas. Son necesarios porque la tecnología no entiende las palabras humanas. Para «explicarle» lo que hay que hacer se necesita un «lenguaje de máquina» especial. Para los usuarios básicos es bastante difícil utilizar constantemente este código, por lo que se han desarrollado sistemas que permiten traducir el habla corriente al binario, familiar para los PC.

La diferencia entre los compiladores y los intérpretes más utilizados es que los primeros generan un archivo listo para usar que se puede ejecutar, mientras que los segundos crean un archivo que funciona únicamente con la ayuda del propio servicio.
