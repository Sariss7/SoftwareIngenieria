INGENIERÍA DE SOFTWARE

Conceptos de Ingeniería

La Ingeniería de Software es el proceso de desarrollo, prueba e implementación de aplicaciones informáticas para resolver problemas del mundo real siguiendo un conjunto de principios de ingeniería y mejores prácticas. Se ocupan del diseñan y desarrollan juegos de computadora, aplicaciones comerciales, sistemas operativos, sistemas de control de red y middleware, por nombrar solo algunas de las muchas trayectorias profesionales disponibles. Los ingenieros de software aplican principios de ingeniería y conocimientos de lenguajes de programación para crear soluciones de software para los usuarios finales.
